## How to Add Links

- `/thought [url]` - Save a single link with AI summary
- `/sync-bookmarks` - Import from Chrome or Safari folder
- `/import-bookmark-file` - Import from exported bookmark file
- `/import-google-news` - Import from Google News saved articles

## Search

Once you have links, use `/find-link [query]` to search.

Examples:
- `/find-link context profiles` - semantic search
- `/find-link tag:claude-code` - filter by tag
- `/find-link category:marketing` - filter by category
- `/find-link recent:7` - last 7 days
