# nano-banana-cli

**Author:** The Focus AI | **License:** MIT

## Summary

A flexible command-line tool and Claude Code plugin that uses Google's Gemini API to generate, edit, and transform images from natural language prompts. Supports batch processing and integrates directly into Claude Code workflows.

## Key Points

- Text-to-image generation and image editing via AI-powered transformations
- Works with multiple Gemini models including faster flash variant
- Batch processing capabilities for automating image workflows
- Integrates as a Claude Code plugin skill for seamless workflow integration
- MIT licensed and designed for accessibility (no complex setup required)

## Why Saved

Relevant as an AI-native tool that combines image generation with CLI automation. Useful reference for workflow automation with Gemini API and Claude Code plugin patterns.
